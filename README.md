## 🚀 Getting Started
Follow these steps to set up the project locally:

### 1. Clone the repo
```bash
git clone https://github.com/StanislavKapytsia/2048_game_js
cd 2048_game_js
```

### 2. Install dependencies
```bash
npm install
# or
yarn install
```

### 3. Run the project locally
```bash
npm start
# or
yarn start
```
